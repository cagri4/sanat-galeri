2. Create `src/app/(admin)/(protected)/layout.tsx` — the auth-checking layout:
```typescript
import { auth } from '@/auth'
import { redirect } from 'next/navigation'

export default async function ProtectedAdminLayout({
  children,
}: {
  children: React.ReactNode
}) {
  const session = await auth()
  if (!session) {
    redirect('/admin/login')
  }
  return <>{children}</>
}
```

CRITICAL: The `await auth()` call in (protected)/layout.tsx is the defense-in-depth layer against CVE-2025-29927. This must NEVER be removed even though middleware also guards admin routes. The middleware can be bypassed with crafted headers; the server-side session check cannot.
